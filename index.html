Peroneal Strength · เสริมเอ็นข้อเท้า

เจ็บเฉพาะ pace เร็ว = เอ็น peroneal ยังไม่แข็งแรงพอรับแรงสูง. โปรแกรมนี้สร้างเอ็นให้กลับมารับ speed ได้. ทำ 3-4 วัน/สัปดาห์ (เน้นวันศุกร์ + ก่อน/หลังวิ่ง). แต่ละท่ามี โน้ตปรับใช้ตามสภาพ — ตึง/ระบมให้พัก, ไม่ตึงให้ทำ. กด ☐ เมื่อทำเสร็จ.

🎯 เกณฑ์กลับไป speed (อย่ารีบ)
  1. ทำโปรแกรม 2-3 สัปดาห์ ให้เอ็นแข็งแรงขึ้นก่อน
  2. ทดสอบ: วิ่ง pace 5:00 สั้นๆ → ถ้าไม่เจ็บ ค่อยขยับ
  3. ค่อยเพิ่ม pace: 5:00 → 4:40 → 4:20 → 4:10 ทีละสัปดาห์
  4. ⚠️ ถ้าหลัง 2 สัปดาห์ยังเจ็บตอน speed → พบนักกายภาพบำบัด
📌 กฎเหล็ก
  1. ทำแล้วไม่เจ็บแหลม (ตึงได้) — เจ็บให้ลดแรง/จำนวน
  2. Eccentric (ช่วงลง/กลับช้าๆ) สำคัญสุด — สร้างเอ็นได้ดีที่สุด
  3. สม่ำเสมอ > หนัก · คลาย(น่อง+ก้น)ก่อน แล้วค่อย strengthen
06

Strength for Runners · โพสทีเรียร์เชน

วันศุกร์. Strength ที่ดีสำหรับนักวิ่งไม่ใช่แค่ squat — แต่คือ hip hinge ที่ฝึกจัดการแรงผ่านสะโพก (glute/hamstring/core) เพื่อรักษาฟอร์มตอนล้า km 30+ และ Running Economy. เน้นฟอร์มถูก > น้ำหนักมาก.

🔑 ทำไม hip hinge สำคัญกับนักวิ่งไกล
  1. ปลายมาราธอนที่พังไม่ใช่หัวใจ/ปอด แต่คือ "สะโพกยุบ ฟอร์มเสีย พลังงานรั่ว"
  2. Posterior chain แข็งแรง = รักษาคุณภาพก้าวได้แม้ล้า = ไม่ faded
  3. ฝึกระบบ "รับ→เก็บ→คืน" แรง (Stretch-Shortening Cycle) = วิ่งประหยัดแรง
  4. ไม่ต้องหนักมาก — 2×/สัปดาห์ก็เห็นผล เน้นฟอร์มถูก
🪑 Active Couch Potato — สำหรับคนทำงานออฟฟิศ (BCEL!)
  1. ซ้อมเช้า 1 ชม. ไม่ลบผลเสียการนั่ง 8-10 ชม./วัน
  2. ลุกขยับ 2-5 นาที ทุก 1-2 ชม. · เดิน 10-15 นาทีหลังมื้ออาหาร
  3. ขึ้นบันไดแทนลิฟต์ · ทำ calf raise/squat เบาๆ ระหว่างวัน
  4. กล้ามเนื้อที่ขยับ = ตัวควบคุมน้ำตาลในเลือด (สำคัญต่อ metabolic health)

⚠️ ช่วงฟื้นตัวตอนนี้ (ขอบเท้ายังไม่นิ่ง 100%)

Zone Guide · 80/20

หลักการทองคำ: ~80% ของ mileage แล่นเบา (Zone 1-2) · ~20% หนัก (Zone 3-5). Zone HR ปรับให้ตรงกับ LT จริง 172 bpm (test มิ.ย.) — ไม่ใช้ตัวเลขทั่วไปอีกต่อไป.

📅 ตัวอย่างสัปดาห์ (Peak phase)

⚠️ กับดักใหญ่สุด
  1. แล่นหนักปานกลาง (Zone 3) "ทุกวัน" = เหนื่อยพอสะสมล้า แต่ไม่เบาพอฟื้น ไม่หนักพอพัฒนา
  2. วันหนัก (Zone 3-5) ต้องตามด้วยวันเบา (Zone 1-2) เสมอ — สลับหนัก/เบา คือหัวใจ
  3. Easy/Long ที่ "รู้สึกช้าเกินไป" มักคือความเร็วที่ถูกต้อง — อดใจไว้
  4. ช่วงฟื้นตัวตอนนี้: เข้ม Zone 1-2 · เก็บ Zone 3-5 ไว้จนขอบเท้าไม่เจ็บ

      RPE Scale · ความเหนื่อย 1-10

      จาก ebook บท 23 — ความเหนื่อยไม่ใช่ศัตรู. ใช้ RPE (ความรู้สึก) ควบคู่ HR/pace. สำคัญ: แยก "เหนื่อยผลักผ่านได้" กับ "เจ็บต้องหยุด".

      ⚠️ เหนื่อย vs เจ็บ (บท 23)
      1. ความเหนื่อย = หนักที่ขา/ปอด ทั่วๆ → ผลักผ่านได้ (ในงานที่ตั้งใจหนัก)
      2. ความเจ็บ = แหลม เฉพาะจุด ที่ข้อ/เอ็น/กระดูก → หยุดทันที อย่าฝืน
      3. เจ็บขอบเท้าตอนเร่ง (ของเจ้า) = สัญญาณ "เจ็บ" ไม่ใช่ "เหนื่อย" → ถอย
